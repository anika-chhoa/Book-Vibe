📚 BookVibe

A dynamic and user-friendly React web application that helps users explore books and manage their reading journey efficiently.

🚀 Live Demo

🌐 Live Site: https://book-vibe-44hlpdy7h-anika-chhoas-projects.vercel.app/
💻 GitHub Repository: https://github.com/anika-chhoa/Book-Vibe.git

📖 Project Overview

BookVibe is a modern React-based book management application where users can explore books, view detailed information, mark books as read, and save books to their wishlist.

The application focuses on creating a smooth and practical reading management experience with persistent data storage using Local Storage.

✨ Features

✅ Interactive and responsive homepage UI
✅ Dynamic routing with detailed book pages
✅ Mark books as Read
✅ Add books to Wishlist
✅ Separate tabs for:

Read Books
Wishlist

✅ Sorting functionality:

Sort by Ratings
Sort by Number of Pages

✅ Persistent data using Local Storage
✅ Reset functionality to clear stored data
✅ Clean and user-friendly interface
